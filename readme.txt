== Installation ==

Via WordPress -
1. Install as a regular WordPress plugin.
2. Upload tr-recent-view to the /wp-content/plugins/ directory
3. Activate the plugin through the 'Plugins' menu in WordPress




Via FTP -

1. Upload .zip file to your WordPress plugin directory and unzip it.
2. Go your Plugins setting via WordPress Dashboard and activate it.
3. Then Publish some post on by go to'tr-recent-view' and 'Add-new' post type with thumbnail
